A Convolutional Neural Network (CNN) is a deep learning model specifically designed for analyzing visual data such as images.
Unlike traditional MLPs, CNNs automatically learn spatial hierarchies of features ‚Äî edges, textures, shapes ‚Äî by using convolutional filters.

CNNs are highly effective for tasks like image recognition, object detection, and image classification.

2Ô∏è‚É£ Why CNN instead of MLP?

MLP treats all pixels independently and ignores spatial relationships.

CNN uses convolutional filters to extract local features like edges and corners.

CNNs have fewer parameters due to weight sharing and local connectivity.

CNNs generalize better and are computationally efficient for image data.

Viva / Oral Questions and Answers:

Q1. What is a CNN?
A Convolutional Neural Network is a deep learning model designed to process and classify image data using convolutional, pooling, and fully connected layers.

Q2. What is convolution?
Convolution is an operation where a kernel (filter) slides over the image to detect features like edges or corners.

Q3. What is the role of ReLU?
ReLU introduces non-linearity and helps prevent the vanishing gradient problem.

Q4. What is the purpose of pooling?
Pooling reduces the spatial dimensions of the feature map and helps make the model more robust to small image translations.

Q5. What is the difference between CNN and MLP?
CNN preserves spatial relationships using convolutions; MLP flattens the input, ignoring spatial structure.

Q6. Why is CNN better for images?
It captures local patterns, requires fewer parameters, and generalizes well for visual tasks.

Q7. What is padding and stride in convolution?
Padding adds extra pixels around the image to preserve size; stride determines how far the filter moves during convolution.

Q8. What loss function is used for image classification?
Cross Entropy Loss.

Q9. What optimizer is used in this experiment?
Adam or SGD optimizer.

Q10. What are the main layers in a CNN?
Convolutional layer, activation function (ReLU), pooling layer, and fully connected layer.
